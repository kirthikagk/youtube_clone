# YouTube Clone - Django + MongoDB Atlas

A YouTube-style video platform built with Django and MongoDB Atlas, featuring video upload, playback, comments, and likes/dislikes functionality.

## Features

- 🎥 Video upload and playback
- 👤 User authentication (signup/login/logout)
- 👍 Like/Dislike system
- 💬 Comments section
- 🔍 Search functionality
- 📱 Responsive design with Bootstrap
- 🗄️ MongoDB Atlas integration

## Prerequisites

- Python 3.8+
- MongoDB Atlas account
- pip (Python package manager)
